# v0.3 evaluation methodology

The study has three sequential stages. First, reproduce and audit v0.2 without modifying its artifacts. Second, calibrate v3 on source-clustered development and validation data, freeze the evaluator, and open holdout only if every upstream gate passes. Third, and only then, run 48 production cases across No Skill, generic business writing, v0.1.0-alpha, v0.2.0-beta, and revised v0.3.0-rc.1.

Two fresh condition-blind judges from model families different from the generation model, and preferably from each other, score each output. Applicability, passage function, hard failure, large category, or total-score disagreements trigger authoritative adjudication. An adjudicator above both primary totals must explain the increase specifically.

Reporting includes full score distributions, paired and category deltas, applicable-category reliability, rank correlation, positive and negative hard-failure agreement, adjudication rate, refusal behavior, regressions, verbosity, ceiling effects, and source-clustered bootstrap intervals with seed 20260720. Same-family or unknown-family judging blocks release.
