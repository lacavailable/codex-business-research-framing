# v0.3 evaluation report

## Decision

Do not tag or release `v0.3.0-rc.1`.

## Completed

The unchanged v0.2 report was recomputed byte-for-byte, evaluator defects were documented, v3 split interfaces and category scoring were implemented, leakage and Unicode controls were added, reconciliation was repaired, private ingestion and copyright safeguards were created, and the balanced public synthetic calibration suite was validated.

## Not run

The public synthetic development/validation run produced 224 primary scores. Primary normalized-score Spearman was 0.712, applicability agreement was 80.9%, negative hard-failure agreement was 60.0%, and 107 of 112 cases (95.5%) required adjudication. This exceeded the preregistered 30% ceiling, so authoritative adjudication was not run and the result is explicitly preliminary.

The private expert calibration, locked holdout, empirical prestige and paraphrase diagnostics, Skill 2.1 revision, repaired 48-case/five-condition production benchmark, and release packaging were not run. They are downstream of failed or unavailable gates. Running them would violate the preregistered sequence.

## Release blockers

- Zero lawfully supplied private passages.
- Zero real qualified reviewer annotations.
- No expert semantic-equivalence verification.
- No evaluator-freeze commit eligible to unlock holdout.
- No demonstrated cross-model-family judging separation.
- No calibrated basis for revising the Skill or running production evaluation.

This is a candid negative gate result, not evidence that the evaluator or Skill has passed calibration.
