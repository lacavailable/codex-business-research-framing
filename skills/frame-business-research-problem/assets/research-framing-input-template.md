# Research Framing Input

Use `unknown` when a field is not yet known. Missing information lowers the maturity ceiling; it must not be invented.

| Field | Readiness | Input |
|---|---|---|
| Research domain | essential | OM, IS, OR, or cross-domain |
| Research objective | recommended | What the study is trying to understand or improve |
| Model objective | essential | Objective function or precise conceptual objective |
| Decision maker | essential | Actor that controls the modeled choices |
| Decisions | essential | Controlled variables or actions |
| States | recommended | System conditions not controlled by the actor |
| Parameters | recommended | Fixed or calibrated inputs |
| Uncertainty | recommended | Random quantities and when they resolve |
| Constraints | essential | Feasibility, capacity, policy, or resource limits |
| Decision sequence | essential | Order of decisions, observations, and outcomes |
| Decision-time information | essential | What the actor knows for each decision |
| Behavioral or operational mechanism | essential | How choices affect outcomes |
| Principal results | optional | Results already established by the analysis |
| Proposed contribution | recommended | Intended theoretical, methodological, computational, or practical contribution |
| Rough business setting | essential | Tentative actor, trigger, and operating context |
| Doubts about the setting | optional | Suspected mismatches or alternative interpretations |
| Supplied evidence | recommended | Sources supporting institutional or empirical claims |
| Target output mode | essential | One of the eight supported modes |
| Target manuscript section | optional | Introduction, setting, implications, or another section |
| Target outlet or reader | optional | Intended scholarly or practitioner audience |
| Length and language requirements | optional | Format, length, and language constraints |

Readiness labels mean:

- **essential**: required for model-grounded fidelity assessment;
- **recommended**: materially improves framing and its maturity ceiling;
- **optional**: useful for the selected mode or final presentation;
- **unknown permitted**: any field may explicitly be unknown during exploratory framing, but essential unknowns prevent an `eligible` finding.
