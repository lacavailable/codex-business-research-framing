# Literature-Grounded Managerial Framing

Apply this layer only after fidelity is `eligible`. If fidelity is `ineligible`, stop and repair the interpretation. If fidelity is `not_assessed`, give provisional options and a missing-information list.

## Build the managerial frame

1. State the central managerial question as a choice owned by a named actor.
2. Explain concrete significance using measurable stakes or a clearly labeled evidence need.
3. Identify a non-obvious structural tension; reject free improvements with no trade-off.
4. Trace the mechanism from the actor's choice to model-supported outcomes.
5. Separate model implications, empirical facts, simplifying assumptions, inferences, and unsupported claims.
6. State applicability conditions, counterconditions, and implementation requirements.
7. Give an actionable implication only when actor, decision, condition, mechanism, and consequence are all explicit.

## Audit fields

Record findings for: managerial question, actor, significance, measurable stakes, structural tension, mechanism, evidence, applicability, actionable implication, counterconditions, and implementation requirements.

Do not use polished urgency, a famous company, or a new industry context as a substitute for evidence or model fit.
