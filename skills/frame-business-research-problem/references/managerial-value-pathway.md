# Managerial-Value Pathway

For managerial, operational, or social value, build the complete chain:

`model result → decision-process effect → operational intermediate outcome → business or social outcome`

Classify every arrow as:

- `model-implied`;
- `logically plausible`;
- `requires operational evidence`;
- `requires empirical evidence`;
- `outside current study`.

Do not skip an intermediate link or claim the final outcome unless every
necessary link is supported. If a link is only plausible, use conditional
language and state the evidence requirement once.

Example:

`faster solution time`
→ `solution available before deadline`
→ `more frequent reoptimization`
→ `potential service or cost improvement`

Observed runtime alone does not establish the later links. A post-deadline
certificate cannot improve a completed decision; retrospective value requires
an explicit future audit, validation, or process-design pathway.
