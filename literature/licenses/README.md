# License-evidence staging policy

This directory contains no third-party license artifacts in v0.2.0-beta.

For any future redistributed source, store independently verified license or
permission evidence here, record its provenance and review date, and ensure it
matches the manifest, checksum, attribution, redistribution status, and
`THIRD_PARTY_NOTICES.md`. A link, abstract, or freely downloadable file is not
license evidence by itself.
